Topics

  • Topics are base building block.
  • [TBD] Topic can be independent. {Alternatively, we can provision for a default view and default group, which will contain all un-categorized topics. A default view and default group must be configurable in DB }
  • [TBD] Topic can be part of a group.


Groups

  • Groups are middle building block.
  • Group can contain list of sub-group, as well as list of topics.
  • Group must be part of a View.

Views

  • Views are top building block.
  • View can contain list of sub-view, as well as list of groups.
